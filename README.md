# Site Apple.

<p>Este projeto é um site que recria a Apple Store, emulando sua UI (User Interface) e UX (User Experience), com o objetivo de apenas desenvolver meu conhecimento e praticar minhas habilidades no desenvolvimento web.
</p>

## Tecnologias Usadas:

1.	HTML5
2.	CSS3
3.	JavaScript
4.	Bootstrap

## Objetivo

<p>O projeto tem como objetivo o aprendizado, saber desenvolver utilizando as tecnologias mencionadas, postar e documentar minha evolução no Github.
</p>
